# Wallpapers

Bundled wallpapers for the **Matrix** Hyprland rice.

## Default

The installer sets **`default.jpg`** unless you pick another image.

## Location after install

Wallpapers are copied to:

```text
~/Pictures/Wallpapers/
```

Waypaper and `set-wallpaper` use that folder.

## Change wallpaper

- **GUI:** `Super + W` (waypaper)
- **CLI:** `matugen-apply-colors ~/Pictures/Wallpapers/your-image.jpg`

## Adding your own

Drop images into `~/Pictures/Wallpapers/` (or into this `wallpapers/` folder and re-run the installer). Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`.

# Keybindings

All keybindings are defined in `~/.config/hypr/modules/binds.lua`.

## Super (Windows key) bindings

| Key | Action |
|---|---|
| `Super + Return` | Open terminal (ghostty) |
| `Super + D` or `Super + R` | App launcher (rofi) |
| `Super + L` | Lock screen (hyprlock) |
| `Super + Shift + L` | Logout menu (wlogout) |
| `Super + Q` | Close focused window |
| `Super + F` | File manager (nautilus) |
| `Super + B` | Browser (brave-origin-nightly) |
| `Super + Shift + B` | Secondary browser (firefox) |
| `Super + C` | Code editor (VS Code) |
| `Super + M` | Music player (spotify) |
| `Super + W` | Change wallpaper (waypaper) |
| `Super + E` | File manager (nautilus) |
| `Super + V` | Toggle window float |
| `Super + Shift + V` | Clipboard history (cliphist) |
| `Super + S` | Toggle scratchpad |
| `Super + Shift + S` | Move window to scratchpad |
| `Super + N` | Open Netflix (web app) |
| `Super + Shift + V` | Virtual machine (virt-manager) |

## Window management

| Key | Action |
|---|---|
| `Super + left/right/up/down` | Move focus |
| `Super + Shift + left/right/up/down` | Move window |
| `Super + 0-9` | Switch workspace |
| `Super + Shift + 0-9` | Move window to workspace |
| `Super + mouse:272` | Drag window (hold left click on Super) |
| `Super + mouse:273` | Resize window (hold right click on Super) |
| `Super + scroll up/down` | Cycle workspaces |

## Media keys

| Key | Action |
|---|---|
| `XF86AudioRaiseVolume` | Volume up (+5%) |
| `XF86AudioLowerVolume` | Volume down (-5%) |
| `XF86AudioMute` | Toggle mute |
| `XF86AudioMicMute` | Toggle microphone mute |
| `XF86MonBrightnessUp` | Brightness up (+5%) |
| `XF86MonBrightnessDown` | Brightness down (-5%) |
| `XF86AudioNext` | Next track |
| `XF86AudioPrev` | Previous track |
| `XF86AudioPlay` / `XF86AudioPause` | Play/pause |

## Screenshots

| Key | Action |
|---|---|
| `Print` | Screenshot area (grimblast) |
| `Super + Print` | Screenshot output |
